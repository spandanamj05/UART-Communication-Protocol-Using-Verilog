# UART Communication Protocol using Verilog

## Overview
This project implements a UART (Universal Asynchronous Receiver Transmitter) using Verilog HDL. The design converts 8-bit parallel data into serial data following the UART communication protocol.

## Features
- UART Transmitter
- Verilog HDL Implementation
- Behavioral Simulation using AMD Vivado 2025.2
- Configurable Baud Rate
- 8-bit Data Transmission
- Start and Stop Bit Generation

## Tools Used
- AMD Vivado ML Edition 2025.2
- Verilog HDL
- Vivado Simulator

## Project Structure
- uart_tx.v – UART Transmitter
- uart_tx_tb.v – Testbench
- Report – Project Documentation
- Simulation Waveforms

## Simulation
The UART transmitter was verified using behavioral simulation in Vivado. The waveform confirms successful serial transmission of 8-bit data.

## Applications
- Embedded Systems
- FPGA Design
- Serial Communication
- IoT Devices
